    • SprintDict

    SprintDict is Android app that allows users to search words and phrases in StarDict dictionaries.

    Functions & Features

    • Search in unlimited number of dictionaries.
    • Access dictionaries on internal or external storage.
    • Enable/disable selected dictionaries.
    • Suggestions while typing a search request.
    • Change the sequence of search results by changing dictionaries' order of succession.
    • Expand/collapse search results by individual dictionary.
    • Change font sizes, zoom in and zoom out.
    • Find a word "on page".
    • Recent search history.
    • Integration with Quick Search Box.
